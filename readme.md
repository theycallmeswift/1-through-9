ReadMe
======
Problem
-------
Using only the numbers 1 through 9, without changing their order insert either a "+", "*", or nothing between each number. Find all the expressions that evaluate to 2002.

Usage
-----
	ruby 2002.rb

Output
------
There are 2 possibilities.

	1*2+34*56+7+89 = 2002
	1*23+45*6*7+89 = 2002
